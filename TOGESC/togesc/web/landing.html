El metodo pedagogico

🧠

Algoritmo SRS

Repite lo que te cuesta y espacia lo que dominas. El motor optimiza cada sesion segun tus tiempos de respuesta y errores recientes.

🎹

Variacion universal

Entrena con multiples timbres y octavas. Aprende la clase de altura (Do, Re, Mi...) y no una frecuencia fija en un solo instrumento.

〰️

Limpieza tonal

Tras cada ejercicio, un cluster breve rompe el anclaje al tono anterior y favorece oido absoluto puro.

Tu hub de entrenamiento

Interfaz pensada para el foco: modos de practica, piano interactivo, estadisticas locales y cuenta opcional para sincronizar progreso entre dispositivos.

  • Repeticion espaciada en el cliente
  • Estadisticas de retencion y notas dificiles
  • Modo velocidad para reflejos rapidos
  • Progreso local siempre; sync Pro opcional

Modos de entrenamiento

Disenados para cada etapa de tu desarrollo auditivo.

La base fundamental. Identifica tonos aislados en cualquier octava.

Intervalos

Acordes

Modo velocidad

Desafia tus reflejos con rafagas de identificacion rapida.
