Contexto
TOGESC (TOne GEneration SCript) es un entrenador de oido absoluto escrito en Python con CLI. Se migra a Flutter para tener UI grafica, soporte movil/desktop, y piano interactivo. El proyecto Flutter ya existe en D:/entrenador_oido/TOGESC/togesc/ (boilerplate). Se sigue metodologia V-model: unit tests por modulo, integration tests entre modulos, system tests al final.
Decisiones tomadas:

State management: Riverpod
Audio: flutter_soloud (buffers PCM en memoria)
Piano: Interactivo (teclas tactiles + texto como alternativa)

FASE 2: SRS Engine (Dart puro) + Note Parser

_scheduleNextReview(note, wasCorrect) -- intervalos learning vs consolidacion
getOverdueNotes(), getLearningNotes(), getStatistics(), getPracticeRecommendations()
Inyeccion de DateTime Function() y Random para testing determinista


lib/services/progress_repository.dart -- Interface + impl JSON con migracion formato antiguo
lib/services/note_parser.dart -- parseNotes(): uppercase, split, normalizar enarmonias

Unit Tests:

test/unit/srs_system_test.dart:

selectNotes con overdue, learning, weighted fallback, notePool
updateAfterResponse correcto: weight baja, ease sube si rapido, consecutive++

Graduacion: 5 consecutivos -> isLearning=false
Regresion: error en consolidacion con interval bajo -> vuelve a learning
Penalizacion wrong_notes externas
_scheduleNextReview para cada rama

FASE 3: Audio Synthesis Engine (paralela a Fase 2)
Archivos:

lib/services/audio_generator.dart -- Port en Dart puro con Float64List:

generateTone(frequency, {duration, instrument}) -- sintesis aditiva + ADSR
_applyAdsrEnvelope(audio, duration, attack, decay, sustain, release)
_generatePinkNoise(duration) -- Voss-McCartney
_generateSweepingTone(duration, fStart, fEnd) -- sweep con fase acumulada
generateCluster({duration}) -- 50 sweeps + pink noise 70/30, envelope nube
getNoteFrequencies(noteNames, {varyOctaves}) -- frecuencias con variacion octava
mixTones(frequencies, {duration, instrument}) -- genera buffer mezclado listo para playback


lib/services/audio_player_service.dart -- Wrapper flutter_soloud:

initialize(), playBuffer(Float64List, sampleRate), dispose()
_float64ListToWavBytes() -- genera WAV header + PCM 16-bit



Unit Tests:

test/unit/audio_generator_test.dart:

generateTone: longitud correcta (sampleRate * duration)
generateTone sine: valores aproximan sin(2pif*t)
ADSR envelope: t=0 ~0, t=attack ~1, sustain ~level
Pink noise: longitud correcta, valores en [-1, 1]
Sweeping tone: longitud correcta, no-zero
Cluster: longitud correcta, dentro de limites clipping
getNoteFrequencies: A4=440Hz, con octave shift se duplica
WAV bytes: header RIFF valido



Fuente Python: audio/generator.py
